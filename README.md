# 📡 Information Coding and Transmission Laboratory

This repository contains practical implementations and laboratory experiments developed as part of the course:

**ICT-4102 – Information Coding and Transmission Laboratory**

---

## 📘 Course Overview

This laboratory course focuses on the practical implementation of core concepts in information theory and communication systems. It emphasizes understanding how data is encoded, transmitted, and affected by noise in real-world communication channels.

The course bridges theoretical knowledge with hands-on programming to build a solid foundation in data transmission and coding techniques.

### Key Topics Covered

- Communication noise analysis  
- Random noise characteristics (mean & variance)  
- Noiseless binary channel simulation  
- Binary Symmetric Channel (BSC)  
- Channel capacity and information theory  
- Source coding techniques  
- Shannon-Fano coding  
- Huffman coding  
- Data compression fundamentals  

---

## 🧪 Laboratory Experiments

This repository includes implementations of the following experiments:

1. **Communication Noise Analysis**  
   - Study and simulation of common noise types (Gaussian, Impulse, Thermal, Random)  

2. **Random Noise Characteristics**  
   - Analysis using different mean and variance values  

3. **Noiseless Binary Channel Simulation**  
   - Ideal channel behavior without noise  

4. **Binary Symmetric Channel Capacity**  
   - Calculation of channel capacity based on error probability  

5. **Shannon-Fano Coding Algorithm**  
   - Implementation of entropy-based coding technique  

6. **Huffman Coding Algorithm**  
   - Optimal prefix coding for data compression
